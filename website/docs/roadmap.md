# Roadmap

Non-binding direction; track execution on [GitHub Issues](https://github.com/wavesMeetU/stratum-map/issues).

## Near term

- Hosted **API reference** parity with shipped TypeScript (this Docusaurus site).
- **Screenshots** and short clips for examples gallery.
- **CI** matrix for `npm test` on Node 20/22.

## Medium term

- Additional **ingest** formats and streaming ergonomics (see repository parser modules).
- **Performance guide** per browser vendor (buffer upload patterns, adapter limits).

## Long term

- Optional **npm publish** automation and semver policy for `1.x`.
- Expanded **worked examples** beyond OpenLayers (minimal hosts still supplying map→clip).

Pull requests that advance items with tests and docs are welcome — see [Contributing](./contributing.md).
